Code $O(logP)+O(P)$ 打表

打表法就是将题目中需要的答案集合提前算出来，存到代码里，根据题目所需取答案，这种方法通常只需要将程序挂着，在表打完后进行加工，最终取答案程序时间复杂度为O(1)，空间复杂度为O(n)(n为答案规模)； ——沃兹·基朔德

P是所有质数的数量。二分查找。

Code TLE $O(n^{1.5})+O(1)$ Force
