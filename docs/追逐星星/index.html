追逐星星

In Think

那天我看到一颗星星，我想把它摘下。

而我只有我的百尺危楼。

我在地面奔跑，我追逐晨昏线的移动。

在奔跑的过程中，我的血液开始沸腾起来。

在奔跑的过程中，我从黑暗的森林中穿过。

在奔跑的过程中，我始终抬头看着那颗星。

我有时会跨过极点，看到天空中曼妙的七彩极光。在这虚幻寒冷的自然幻象中，只有我的心还是滚烫。

流动的空气，星星闪动。

我看到了璀璨的星海，可我只注意到那颗特别的星星。

风很大。淹没了一切声响。

我跑过平原和山川，我似乎感到远方那颗星，隐隐的势。

我拼命地奔跑啊！

我的眼泪也悄悄地流淌。

我看着那若即若离的星光。忽然心中一动。

我心中一痛。

时间流走了。

我的年岁流走了。

啊，我也什么也不剩下。

我看着模糊的星光。终于从中看到我逝去的影子。

是什么时候？

追逐星星成为了一种让我畏惧的行为。

是什么时候?

理性战胜了我。

那时我便知晓，我已不再是可以追逐星星的人。

啊，这恒久远的深空。

啊，这恒久远的寂静。

我停下了脚步。

我感到久违的惯性，我感到一层又一层上涌的虚脱。

我累了。

我再也不要追逐星星。

在这星海里面，我早已迷失。

山顶的风好冷。

我遥看向无限延伸的地表。

终于有一天我明白，我脚上踩着的，才是唯一属于我的星星。

在我奔跑的过程中，我已将足迹遍撒星星的表面。

嗯。

我将这颗星星上闭关。

也许有一天，当我理解了我的星星。

它也会变成一颗。

无比炙热。

无比闪亮的。

恒星。
