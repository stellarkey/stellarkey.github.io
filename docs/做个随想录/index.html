做个随想录

In Work

受到 Hitokoto 一言 的启发。前几日看到一个博主：智伤帝。动了念头。

目标：一个可以动态维护的文字系统，前端进行展示和传输，后端进行存储和处理。

创建数据库（废止）

使用 SQLite 创建所需的数据库存储系统。

前端的选择

  • typed.js 打字机
  • prisma.js 数据库操作（废止）

最终方案

为了简单，直接本地编辑文章，网页端动态读取并转换成句子列表，然后 shuffle 并展示出来。

效果展示：https://vel.wiki/
