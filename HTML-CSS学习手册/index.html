XML 简介

XML（eXtensible Markup Language，可扩展标记语言）。作为一个完整的标记语言，相比JSON的冗余度更高，解析器设计更复杂。但是数据检索和复用上XML更有优势。

XML和HTML很像，但是XML被设计用来传输和存储数据，HTML用来显示数据。

XML没有预定义的标签，而HTML的标签都是标准化的。标签是由 XML 文档的创作者自行发明的。如下所示：

其中的<to>, <from>标签并未在任何XML标准中定义过。

XML将数据内容和HTML布局分离，通过JavaScript的简单调用代码读取XML来更新HTML中的数据内容。
